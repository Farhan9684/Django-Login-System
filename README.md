# Django Login System  

## Features ✅  
🔐 User Signup & Login  
🏠 Protected Home Page (requires login)  
🚪 Logout Functionality  

## Installation 🛠️  

1️⃣ **Install the latest version of Python**  
```sh
python --version
```
2️⃣ **Install Django**
```sh
pip install django
```
3️⃣ **Apply Migrations**  
```sh
python manage.py migrate
```
4️⃣ **Run the Server**  
```sh
python manage.py runserver
```
📌 Visit **`http://127.0.0.1:8000/`** 🎉
